Delta Trading Stock Application

DeltaTradingStocks is a comprehensive system for stock market analysis, featuring a robust Kotlin backend and a connected real-time client. The backend connects to financial data sources via WebSockets and APIs, ingesting both live and historical market data for specified tickers like NVDA, TSLA, and MSFT. It performs various calculations, including Average True Range (ATR), volatility metrics, and session analysis, to derive valuable insights. This processed data and real-time updates are efficiently delivered to the client application, which utilizes JCEF (Java Chromium Embedded Framework) to present a beautiful and responsive user interface.

Mining (Minecraft)

This project is a custom Java plugin developed for Minecraft servers, designed to enrich the gameplay experience with a focus on mining. It introduces advanced mining mechanics, including unique ore types with specific properties, diverse pickaxe variations, and an intelligent block respawn system to ensure resource availability. The plugin also features a custom mob system with new enemy types, configurable loot tables, and manageable mob spawners, alongside an item system with rarity tiers and a forge for crafting/upgrading. Player progression is enhanced through skill development and ranking systems, with all crucial data like player stats and spawner configurations persistently stored via database integration, creating a more dynamic and engaging server environment.

MapleStory Timer

This project is a simple overlay application designed for the game MapleStory to help players farm items more effectively. It features a customizable timer that alerts the user after a 140-second farming period, followed by a 25-second window for collecting drops, ensuring no items are lost. The overlay includes visual and audio cues and offers a user-friendly interface that can be easily moved and resized on the screen for optimal placement during gameplay.
